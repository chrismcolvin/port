EthioFaith.org

A Christian special education school for Ethiopia.

Challenge

Create and deploy a fundraising website for EthioFaith, a non-profit organization building a special education school in Ethiopia.

Approach

EthioFiath needed an easy to use website that would be up quick, look good, and provide an easy to find and use donation tool. For this, I used Squarespace to get a site up quickly without having to worry about building a custom site.

Create

Using Squarespace, design was simple and fast. EthioFaith had a website up and running in no-time.
